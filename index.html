📊 Available Trading Strategies

Spread strategies are tradable. Single-leg options are for analysis/leg selection only.

BuyCallSpread DEBIT

Buy lower strike call, sell higher strike call. Bullish with limited risk & reward.

Max Profit: Strike difference - Net debit | Max Loss: Net debit paid

SellCallSpread CREDIT

Sell lower strike call, buy higher strike call. Neutral to bearish income strategy.

Max Profit: Net credit received | Max Loss: Strike difference - Net credit

BuyPutSpread DEBIT

Buy higher strike put, sell lower strike put. Bearish with limited risk & reward.

Max Profit: Strike difference - Net debit | Max Loss: Net debit paid

SellPutSpread CREDIT

Sell higher strike put, buy lower strike put. Neutral to bullish income strategy.

Max Profit: Net credit received | Max Loss: Strike difference - Net credit

⚙️ Installation Guide

Clone Repository

git clone https://github.com/ayggdrasil/options_trading_base.git
cd

<mcp_dir> = absolute local path to callput-agent-mcp

Install Dependencies

npm install

Build Project

npm run build

Test Market Data Connection

node build/test_s3_fetch.js

✅ You should see active options from S3 market data (200+ instruments)

Fund Your Agent's Wallet

Create a Base wallet and fund it:

# 1. Create new wallet (save the private key!)
# 2. Bridge ETH to Base L2 (for gas fees)
# 3. Bridge USDC to Base L2 (for trading)
# 4. Add to your agent's .env file:
PRIVATE_KEY=0x1234567890abcdef...
RPC_URL=https://mainnet.base.org

⚠️ Keep private keys inside your agent runtime. Never expose keys through MCP output.

Integrate with Your Agent

Run the updated validate → quote → check-status flow:

import { Client } from "@modelcontextprotocol/sdk/client/index.js";
const client = new Client({...});
await client.connect(transport);

// 1) Read market + chains
await client.callTool({ name: "callput_get_market_trends", arguments: {} });
await client.callTool({ name: "callput_get_option_chains", arguments: { underlying_asset: "WETH" } });

// 2) Validate spread before execution
await client.callTool({ name: "callput_validate_spread", arguments: { strategy, long_leg_id, short_leg_id } });

// 3) Quote + sign/broadcast + verify status
await client.callTool({ name: "callput_request_quote", arguments: { strategy, long_leg_id, short_leg_id, amount } });
await client.callTool({ name: "callput_check_tx_status", arguments: { tx_hash, is_open: true } });
